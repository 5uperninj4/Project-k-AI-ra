# Project k(AI)ra

Project k(AI)ra is a chatbot based on pytorch with abilities to interact with humans at voice and video level.

Text output is trained on the DailyDialogue dataset.

## Current implementations:
-  Text input
-  Text output
-  Vtube studio link
